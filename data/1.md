# IT Infrastructure and Corporate Security Guidelines

This document outlines the security requirements for all employee devices, remote connections, and internal system access. 

## Device Protection and Clean Desk Policy
All company-issued laptops must be encrypted using BitLocker (Windows) or FileVault (macOS). Screens must be set to lock automatically after 3 minutes of inactivity. When leaving your desk, you must manually lock your computer (Win+L or Cmd+Ctrl+Q). Do not write passwords on sticky notes attached to monitors.

## Passwords and Credentials
Multi-Factor Authentication (MFA) is mandatory for all tools. Our primary MFA provider is Okta. Passwords must be rotated every 90 days. The default password for setting up newly issued devices is `TempPassword2026!`. 

## Remote Work and VPN Access
When working from public locations (cafes, airports), employees must connect via the GlobalProtect VPN. The address is `vpn.corporate-hub.com`. The IT emergency contact number is 555-0199 (Ext. 4).
